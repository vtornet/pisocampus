# Iconos PWA

Para generar los iconos de la PWA, puedes usar herramientas como:

1. **PWA Asset Generator**: https://www.pwabuilder.com/imageGenerator
2. **Favicon Generator**: https://favicon.io/
3. **RealFaviconGenerator**: https://realfavicongenerator.net/

## Iconos necesarios:

- `icon-192.png` - 192x192px
- `icon-512.png` - 512x512px
- `apple-icon.png` - 180x180px
- `favicon.ico` - 48x48px (o multiple sizes)

## Colores de la marca:

- Primary: #2563eb (blue-600)
- Text: #0f172a (slate-900)
- Background: #ffffff

Para empezar, puedes usar iconos placeholder simples con las iniciales "AE" sobre fondo azul.
